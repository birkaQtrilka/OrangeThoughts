Internship at Tible

#Angular

Description

  In this internship I was responsible for the front-end of a dashboard application. It was made for Statiegeld Nederland, a Dutch organization that promotes the return of bottles and cans. The web app was made with Angular and TypeScript. It features data visualization with charts and graphs, user authentication, data filtering using custom form fields.
  The main challenge was to work with an existing codebase and to learn new technologies in a short period of time. I had to learn Jira, Confluence, Jenkins, SpringBoot and other tools used in the company.

Time spent:

 6 months
