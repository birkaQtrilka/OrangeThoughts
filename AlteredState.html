Altered State

Description

In this game jam, I worked in a team of 6. I was responsible for the card dragging mechanic, the card management system, the procedural map generation and the sound manager. The sound manager spawns SoundSource components via an object pooling system. I always try to find ways to structure my code with design patterns, but I understand that sometimes it's better to not force it.
