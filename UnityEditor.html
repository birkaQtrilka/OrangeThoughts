Description

Product

  I made a Unity tool that allows you to spawn worlds through the wave function collapse algorithm. It's a scriptable object with a custom editor in which you can add tiles with varying socket types.

My Contributions

  In this project, I learned how to use the unity editor API and also how to decouple data and behavior. Also, I learned what are serialized properties and how to save data changed in the editor.
  The main challenge was to serialize the data and save it after every change. On top of that, the editor has some performance issues that I need to solve.
