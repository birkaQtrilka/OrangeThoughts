Ro-Bombs

Description

In this project I worked in a team of 6. I was responsibe of making and juicing the 4 out of 8 minigames for the game, I made a difdiculty system where each minigame has a custom way of getting harder.
Also, I planned the architecture of the project as well as made a designer friendly interface by integrating the low level engine provided by the university with Tiled, which is an interface that returns the data as xml files.

What is gxp?

It's a low level engine that provides basic 2D physics, rendering, sound, an inheritence based GameObject system and it parses the xml data from Tiled. There is no scene loading, data saving or UI system which I basically made from scratch.
