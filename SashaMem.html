Description

Game

  A point and click story game with diary sections where you have to make the right choices to not spiral into a bad lifestyle.

My Contributions

  In this university project, we were a team of 6. I was responsible for building the framework for our game. I made a game object system with the component pattern, so we could reuse code. I also made an animation component, transform component, playerMovement component, a simple rectangle collider, etc.
  The challenge I encountered was to how make the transform component work. I didn't have much knowledge of matrices and space, so I really had to do some research.

Soft Skills Development

  I was the scrum master and used Trello (a planning tool) to give each team member tasks that we agreed on as a team. We had daily meetings where we discussed what we each did before and what was blocking the progress. In the project, I had to figure out how to give tasks to a less experienced engineer. I also had to explain in detail how my partner should approach the problem, for instance, "if you want to create the second scene, use the transform component to position the sprites and use my player object code from the previous scene to add the player...".
